Previous error code: E0011 AMP overvoltage (P-N)

 

 

1.1.55.1. Outline

 

Direct current voltage (P-N) of Servo Drive Unit that drives the motor exceeded the set value.

 

 

 

1.1.55.2. Causes and examine methods

 

Abnormality with the path that detects the PN voltage drop from the diode module or with the PN discharge circuit

 

< Case: Error always occurs even when the motor is off >

(1) Check the cable related to the overvoltage error detection.

n  Please replace CNSGC cable and test it.

 

(2) Please examine the components that are related to the overheat error detection.

n  Please replace BD530/BD531 board and test it.

n  Please replace a servo drive unit and test it.

 

 

(1)Check the cable related to the overvoltage error detection.

 

n  Replacement of CNGSC and examine CNSGC

Please replace CNGSC cable with a proper one and if an error does not persist, cable connection fault caused an error. Please replace the CNGSC cable with new one and use.

 

(2)Please examine the components that are related to the overheat error detection.

AMP overvoltage error is detected by the servo drive unit when the DC power (P-N) to the servo drive unit exceeds the set level. Detected error will be sent through CNSGC cable to be handled by BD530/BD531 board.

 

n  Replacement of BD530/BD531 and examine

Please replace BD530/BD531 with a proper one and if an error does not persist, the board malfunction caused an error. Please replace the BD530/BD531 with new one and use.

 

n  Replacement and inspection of servo drive unit

ик  Hi5a-S controller : Medium size SD3X3Y, Small size SD3A3D

ик  Hi5a-N controller : Medium size SD1L2C, Small size SA3A3D
